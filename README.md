# **AnimationMenu**
动画菜单viewgroup
## 使用方法

build.gradle文件中添加:
```gradle
allprojects {
    repositories {
        maven { url "https://jitpack.io" }
    }
}
```
并且:

```gradle
dependencies {
     implementation 'com.github.ashLikun.AnimationMenu:{latest version}'//没有databind
}
```
